## ## コンセプト
- 最近ブームの日本流スパイスカレーを効率よく探し、食の記録や感想の共有が可能 

## 制作のきっかけ
- 関西に住んでいた際、毎週のように大阪のスパイスカレー激戦区に足を運ぶ程、スパイスカレーにハマり、この美味しさや感動を共有できる場所が欲しいと思っていた 
- 食べログ等でカレーを検索すると洋風カレーやチェーン店などもヒットしてしまい、目的の日本流スパイスカレーをピンポイントで探すのに苦労した経験があった
